# TRAE Demo Wall项目概要设计 v0.6

## 1. 版本目标
v0.6 在 v0.5 的基础上进行安全加固、数据完整性提升和运营能力增强，修复关键安全漏洞，完善控制台数据统计，扩展城市字典覆盖范围。

## 2. v0.6 核心变更

### 2.1 安全加固
- **Broken Access Control 修复**: 后台 API 添加基于 `roleCode` 的服务端鉴权
- **Hidden Work Disclosure 修复**: 前台作品详情接口添加状态校验
- 普通用户无法访问管理后台接口和数据

### 2.2 数据完整性
- **控制台概览统计修复**: 使用数据库最新记录时间作为统计基准，解决时间偏差问题
- **城市字典扩展**: 从 15 个城市扩展到 344 个城市，覆盖中国全部省级行政区

### 2.3 用户体验优化
- 提交/编辑作品时的错误提示更加具体，显示具体字段名和错误信息
- 富文本编辑器在编辑模式下内容同步修复

## 3. 安全设计

### 3.1 后台 API 鉴权

**受影响接口**:
| 接口路径 | 所需权限 |
|---------|---------|
| `/api/console/works` | admin/root |
| `/api/console/overview` | admin/root |
| `/api/console/cities/stats` | admin/root |
| `/api/users` | admin/root |
| `/api/roles` | admin/root |
| `/api/logs/auth` | admin/root |
| `/api/logs/operations` | admin/root |

**鉴权实现** (`src/lib/auth.ts`):
```typescript
export function hasAnyRole(user: AuthUser | null, requiredRoles: string[]): boolean {
  if (!user) return false;
  return requiredRoles.some(role => user.roles.includes(role));
}

export function isAdmin(user: AuthUser | null): boolean {
  return hasAnyRole(user, ['admin', 'root']);
}
```

**API 使用示例**:
```typescript
export async function GET(req: NextRequest) {
  const user = await getAuthUser();
  if (!isAdmin(user)) {
    return NextResponse.json({ error: 'Forbidden: Admin access required' }, { status: 403 });
  }
  // ... 业务逻辑
}
```

### 3.2 作品可见性控制

**前台详情接口** (`/api/works/[id]`) 鉴权逻辑:
```
未登录用户/普通用户 → 只能查看 auditStatus=1 && displayStatus=1 的作品
作品作者           → 可以查看自己的任何作品
管理员(admin/root)  → 可以查看任何作品
```

**实现代码**:
```typescript
const isOwner = currentUser && work.userId === currentUser.userId;
const isAdminUser = isAdmin(currentUser);
const isApproved = work.statistic?.auditStatus === 1;
const isVisible = work.statistic?.displayStatus === 1;

if (!isOwner && !isAdminUser && (!isApproved || !isVisible)) {
  return NextResponse.json({ error: 'Work not found' }, { status: 404 });
}
```

## 6. 前端实现更新

### 6.1 富文本编辑器修复

**问题**: 编辑模式下 `useEffect` 导致内容被覆盖

**修复方案**:
```typescript
const isInitialMount = useRef(true)
const previousValueRef = useRef<string | null>(null)

useEffect(() => {
  if (!editor || value === undefined) return
  
  // 跳过首次渲染（内容已通过 useEditor 设置）
  if (isInitialMount.current) {
    isInitialMount.current = false
    previousValueRef.current = value
    return
  }
  
  // 只在外部值真正变化时更新
  if (value !== previousValueRef.current) {
    editor.commands.setContent(value)
    previousValueRef.current = value
  }
}, [value, editor])
```
